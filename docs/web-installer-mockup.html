Make Claude an expert in your tech stack.

cc-praxis is a curated collection of Claude Code skills for professional software development. Where Claude Code gives you a capable generalist, cc-praxis gives you a disciplined collaborator that knows your conventions, enforces your quality standards, and chains the right actions together automatically — every session, every project.

🔗

Skills that chain together

Skills are designed as a system. ts-dev invokes ts-code-review, which escalates to ts-security-audit for auth code. The right workflow runs without you orchestrating it.

🧠

Project-aware by design

One CLAUDE.md declaration sets the project type. Every skill reads it — commits route correctly, health checks pick the right categories, docs sync automatically.

⚡

Zero overhead when idle

Installed skills contribute only their description (~30 words) to your context window. The full skill loads only when triggered — so having 33 skills costs almost nothing.
